# Browse google analytics query explorer

This function opens the Google Analytics Query Explorer in a web
browser.

## Usage

``` r
ga_query_explorer()
```

## Details

It allows users to explore and test Google Analytics queries
interactively. This is useful for developers and analysts who want to
experiment with different queries and see the results in real-time.

## Note

This function is intended for users who have a Google Analytics account
and want to explore the API capabilities. It does not return any data or
perform any analysis. You need to have a Google account with google
analytics access and will be prompted to log in if you are not already.
